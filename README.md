# shinyauth
The Dockerfile for the shinyauth R environment used in the [Shiny Developer with AWS](https://university.business-science.io/p/expert-shiny-developer-with-aws-course-ds4b-202a-r/) course. Installs:

- shiny-verse:latest
- shinyWidgets
- shinythemes
- shinyjs
- mongolite
- jsonlite
- config
- remotes
- tidyquant
- plotly
- shinyauthr
